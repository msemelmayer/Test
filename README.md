# Tic Tac Toe

A simple browser-based Tic Tac Toe game built with HTML, CSS, and vanilla JavaScript.

## Features

- **2-player mode** — take turns on the same screen
- **vs CPU** — toggle an unbeatable AI opponent (minimax algorithm)
- **Scoreboard** — tracks X wins, O wins, and draws across rounds
- **Restart / Reset Score** — manage games without losing the session score

## How to Play

1. Download or clone this repository
2. Open `tictactoe.html` in any modern web browser
3. Click a cell to place your mark
4. Toggle **vs CPU** to play against the AI

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (no dependencies)
